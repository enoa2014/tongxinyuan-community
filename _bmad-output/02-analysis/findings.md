# 调研与成果：同心源项目分析

## 2025-2026 战略转型分析

### 1. 核心背景与挑战
- **行业阵痛 (2024-2025)**:
    - **根本原因**: 并非执行违规，而是公募基金会为规避异地监管风险（新《慈善法》“穿透式”管理）。
    - **结果**: 采取“一刀切”策略，清理助医类合作项目，导致同心源面临资金流中断。
- **战略转型**: 从“平台依赖”转向“自主生长”。
    - **旧模式 (脆弱)**: 流量红利 + 外部公募资质 + 被动接收。
    - **新模式 (韧性)**: 私域流量 + 品牌月捐 + 多元造血 + 合规透明。

### 2. 2025年度关键数据
- **核心定位**: 为异地求医的大病患儿家庭打造“异乡的家”。
- **量化成果**:
    - **受益人次**: 5,744 人次 (大病患儿家庭)
    - **住宿服务**: 2,048 天 (运营医科大一附院/区医院附近2个小家)
    - **困境女童**: 受益 400 人
    - **助学金**: 117 名
    - **生命关怀**: 5个临终家庭，79人次志愿者参与培训。

### 3. 2026重点规划：关爱异地求医大病儿童家庭社区支持中心 (新模式)
- **愿景升级**: 从单一住宿点 -> “家以外的家” (复合功能社区中心)。
- **四大功能模块**:
    1.  **生活支持**:
        -   设施: 爱心物资站 + 共享厨房。
        -   价值: 降低生活成本，通过烹饪家乡菜缓解乡愁，重建社交链接。
    2.  **喘息服务**:
        -   内容: 家长心理疏导 + 互助网络。
        -   手段: 社工专业陪伴、艺术疗愈。
    3.  **儿童康乐**:
        -   内容: 绘本阅读 + 游戏治疗。
        -   目标: 防止长期就医导致的心理发展脱轨。
    4.  **生命教育**:
        -   **亮点**: 社区化安宁疗护。
        -   目标: 提升家庭面对生死议题的韧性 (与临床医疗互补)。

### 4. 志愿者与人力体系
采用金字塔式人才培养结构：
- **Level 1 (普适型)**: 后勤支持 (搬运、保洁、接待)。
- **Level 2 (康乐型)**: 需心理/防感培训，负责陪伴、阅读、游戏。
- **Level 3 (专业型)**: 医护、心理师、律师，负责核心的生命教育与安宁舒缓。

### 5. 运营管理机制
- **需求评估**: 严格核实病情证明与经济状况。
- **建档立卡**: 确保资源流向最紧迫群体。
- **物资闭环**: 定期举办“爱心市集”，按需领取，维护尊严。

# 旧版网站地图 (Legacy Site Map)

基于数据库分类表 (`txy_article_categories`) 及控制器分析构建。

## 1. 网站导航结构 (基于内容分类)

通过 Nested Set 模型 (左值/右值) 重建的层级结构：

*   **首页 (Home)**
    *   **新闻资讯**
        *   [列表] 机构动态 (cate_id: 30)
        *   [列表] 媒体报道 (cate_id: 31)
    *   **关于我们**
        *   [单页] 机构介绍 (cate_id: 14)
        *   [单页] 项目介绍 (cate_id: 15)
        *   [单页] 项目点 (cate_id: 16) - *包含: 万秀、北湖、兴宁中学服务点*
        *   [单页] 联系我们 (cate_id: 17)
    *   **支持我们**
        *   [单页] 全职招聘 (cate_id: 33)
        *   [单页] 志愿者招募 (cate_id: 34)
        *   [单页] 我要捐款 (cate_id: 35)
        *   [单页] 合作伙伴 (cate_id: 36)

## 2. 功能页面结构 (非 CMS 内容)

### 用户系统
*   **注册** (`/register`)
*   **登录** (`/login`)
*   **找回密码** (未实现/仅占位)

### 用户中心 (`/account`)
*   **个人首页** (Dashboard)
*   **资料管理**
    *   修改头像 (`changeface`)
    *   修改资料 (`changeinfo`)
    *   修改密码 (`changepwd`)
*   **我的互动**
    *   我的话题 (`mytopics`)
    *   我的评论 (`mycomments`)

### 社区互动
*   **论坛/话题广场** (`/topics`)
    *   发布话题
    *   话题详情
*   **文章详情页**
    *   文章内容
    *   发表评论功能

## 3. 后台管理结构 (`/admin`)

*   **系统管理**
    *   管理员管理
    *   角色/权限管理
    *   登录日志/操作日志
*   **内容管理 (CMS)**
    *   文章发布/编辑/删除
    *   文章分类管理
    *   图片新闻管理 (首页轮播)
*   **用户与审核**
    *   会员管理 (查看/停用)
    *   评论审核
    *   话题审核
    *   缓存管理
